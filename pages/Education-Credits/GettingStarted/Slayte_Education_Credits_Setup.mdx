## Introduction


This article outlines, how to create Education Credits in Slayte, which can be used in Slayte Events Module, to be assigned to sessions, then claimed by the attendees and generate a certificate, or register them in Impexium custom tables.


 


## Requirements


* An [Education Credits Program in Impexium](https://help.impexium.com/Content/event_education%20credits.htm?Highlight=education%20credits) (Optional if integration will be used).
* An Education Credits Provider (Optional if integration will be used to register credits in Impexium).
* [An event in Slayte](https://docs-for-customers.slayte.com/hc/en-us/categories/4412140122259-Conference-Manager), that will have claimable sessions for the education credits.
